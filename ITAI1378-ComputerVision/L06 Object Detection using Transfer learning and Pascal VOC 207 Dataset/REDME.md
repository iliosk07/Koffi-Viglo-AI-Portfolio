# CNN-Based Image Classification: Chihuahua or Muffin Lab
This assignment is about learning object detection using TensorFlow and the Pascal VOC dataset. Instead of just telling what an image is, object detection also shows where the objects are with boxes around them. I used a pre-made Jupyter Notebook and ran it in Google Colab (or SageMaker) so I could use a GPU for faster processing. I followed the steps to set up the environment, checked that the GPU or TPU was available, and loaded the SSD MobileNet V2 model to detect objects in images from Pascal VOC 2007.
